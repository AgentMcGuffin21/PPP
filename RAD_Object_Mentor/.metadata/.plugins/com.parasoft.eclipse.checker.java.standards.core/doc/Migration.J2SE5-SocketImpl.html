New abstract methods added to SocketImpl [Migration.J2SE5-SocketImpl-3]

DESCRIPTION 
As of 1.4 the Socket API added a abstract methods to the
SocketImpl abstract class. Because of the new method,
a subclass of SocketImpl created prior to 1.4 fails to
compile because there is no implementation for the new 
methods. See
http://java.sun.com/j2se/JM_White_Paper_R6A.pdf
