If you are using Rational Application Developer (as opposed to a 
generic Eclipse distribution), use the following setup steps.

1) Delete the following directory: 
    c:\workspaces\RAD_Object_Mentor
2)  Unzip RAD_Object_Mentor.zip into c:\workspaces, resulting in
    c:\workspaces\RAD_Object_Mentor
3) Open RAD, select workspace: 
    c:\workspaces\RAD_Object_Mentor
4) Switch to the "Java" perspective
5) Close all projects except Exercise1
6) Open project Exercise1
7) Run all the tests
   * Right click the source directory
   * Select Run/JUnit Test
   * See 13 of 13 tests pass
